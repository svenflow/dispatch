📥 Message Sources

MessagesReader

Polls macOS chat.db every 100ms for new iMessages. Handles text, attachments, audio transcriptions, reactions, and group detection.

SignalListener

Background thread connected to /tmp/signal-cli.sock JSON-RPC socket. Receives push notifications from signal-cli daemon.

TestMessageWatcher

Development testing via JSON files dropped in ~/.claude/test-messages/. Enables testing without real messaging.

Sven App API

HTTP server on port 8080 for iOS voice assistant app. Creates sessions tagged sven-app: in registry.

⚙️ Manager Daemon

Manager

Main orchestrator in manager.py. Coordinates polling, routing, health checks, and lifecycle management. Single entry point for the system.

ContactsManager

SQLite-based contact lookups from macOS Contacts.app. O(1) phone→tier resolution via in-memory cache.

SessionRegistry

Persistent JSON file at state/sessions.json. Maps chat_id → session metadata (name, tier, source, model, timestamps).

IPCServer

Unix socket at /tmp/claude-assistant.sock. Handles CLI commands: inject, kill, restart, status, set_model.

ReminderPoller

Checks macOS Reminders.app for due items. Supports one-time and cron-based recurring reminders with contact targeting.

🧠 Agent Sessions

SDKBackend

Manages all ClaudeSDKClient sessions. Handles creation, injection, health checks, idle reaping, and graceful shutdown.

SDKSession

Per-contact Claude agent wrapper. Async message queue, concurrent send/receive loops, mid-turn steering support.

Message Queue

Each session has an asyncio.Queue. Messages injected immediately; Claude sees them as UserMessages between tool calls.

BackendConfig

Source-specific strings in backends.py: CLI commands, prefixes, labels. Adding new messaging backend = one entry.

🔧 Health & Recovery

Watchdog

LaunchAgent running every 60s via bin/watchdog. Detects daemon crashes, spawns healing Claude with --dangerously-skip-permissions.

Fast Health Check

Every 60s: regex scan of session transcripts for fatal errors (400 API, context overflow, buffer crash). Auto-restarts affected sessions.

Deep Health Check

Every 5 min: Haiku-based analysis of recent messages for subtle failure modes. Catches issues regex misses.

Idle Reaping

Sessions inactive >2 hours are killed to free resources. Master and background sessions exempt.

🔌 Supporting Services

signal-cli Daemon

Java process with --receive-mode on-connection. Provides JSON-RPC socket for bidirectional Signal messaging.

Memory Search Daemon

Bun-powered TypeScript service on port 7890. SQLite FTS for semantic memory search across contacts.

Gemini Vision

Async image analysis via gemini-3-pro-image-preview. Describes attachments and injects results into sessions.

LaunchAgent

com.dispatch.claude-assistant.plist ensures daemon auto-starts on boot. Uses launchctl for clean restarts.

👥 Contact Tier System

Admin

Full access, bypass permissions, all tools. Owner of the system.

Wife

Full access + warm tone. Extra care in responses.

Family

Read-only mutations. Changes need admin approval. 50 turn limit.

Favorite

Restricted tools, no file writes. Only whitelisted Bash commands. 30 turn limit.

Bots

Same as favorites + loop detection. Won't respond to circular conversations.
